◆ NumFramesDirty

int RenderItem::NumFramesDirty = gNumFrameResources

Dirty flag indicating the object data has changed and we need to update the constant buffer. Because we have an object cbuffer for each FrameResource, we have to apply the update to each FrameResource. Thus, when we modify obect data we should set NumFramesDirty = gNumFrameResources so that each frame resource gets the update.
